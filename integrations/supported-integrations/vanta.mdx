This is a special integration for [Access Reviews](/access-reviews/). It uploads access control evidence to the (documents section)[https://app.vanta.com/documents?standard=soc2#access-reviews] at Vanta.

## Usage

After this integration has been added, evidence is automatically uploaded when an access review campaign is completed. Furthermore, you can upload historical campaigns via click on the upload icon. There is an indicator when evidence was uploaded.
